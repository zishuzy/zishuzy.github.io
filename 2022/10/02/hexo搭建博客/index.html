4.2 deploy 部署

先安装扩展npm install hexo-deployer-git

deploy配置部署的相关配置，其中的

  1. type: 声明部署类型
  2. repo: 声明远程地址
  3. branch: 声明分支名

4.3. excerpt 首页内容隐藏

先安装扩展npm install hexo-excerpt

  1. depth: 深度，可以理解为行

5. hexo 常用命令

  1. hexo clean: 清空生成的博客
  2. hexo g: 生成博客
  3. hexo s: 开启本地服务(localhost:4000)，查看博客
  4. hexo d: 部署博客，上传到git仓库中
